Код: S2-03 — Strengths/Dev areas генератор

Цель
- Простые пороги формируют 3–5 пунктов на раздел, стабильный порядок.

Описание
- Правила формирования сильных сторон и зон развития на основе значений метрик и/или их веса. Пороговые значения и количество элементов ограничены; сортировка стабильная по значению и коду.

Шаги
- Выделение топ‑метрик как strengths; нижних — как dev areas; генерация кратких причин/действий.

Тестирование (обязательно)
- Проверка количества (≤5), порядка и отсутствия дубликатов; воспроизводимость результата.

AC: ≤5/5; повторяемость. Зависимости: S2-02.
