Код: VPN-01 — WireGuard entrypoint

Цель
- Поднимать `WG_INTERFACE` из `WG_CONFIG_PATH` до старта приложения; падать при ошибке VPN.

Описание
- Скрипт/entrypoint поднимает интерфейс WireGuard в контейнере, проверяет его состояние и только затем запускает приложение. При ошибках — фейлить запуск для предотвращения утечки трафика в обход VPN.

Шаги
- Вызов `wg-quick up <iface>` с конфигом по пути; ожидание интерфейса; обработка ошибок.

Тестирование (обязательно)
- Имитация валидного/невалидного конфига; проверка, что приложение не стартует без активного интерфейса при `VPN_ENABLED=1`.

AC: при `VPN_ENABLED=1` интерфейс up, при сбое — 503/не стартует. Зависимости: S1-11.
