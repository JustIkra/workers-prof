Код: S2-01 — Метрики (ручной ввод)

Цель
- Модели/миграции `metric_def`, `extracted_metric` + ручной ввод/редактирование.

Описание
- Ввести словарь метрик (`MetricDef`) и таблицу извлечённых значений (`ExtractedMetric`) с привязкой к отчёту и метрике. На UI — формы ручного ввода/редактирования значений в диапазоне [1..10] с валидацией.

Шаги
- Миграции/модели; API для чтения/записи метрик отчёта; простая вёрстка форм.

Тестирование (обязательно)
- Уникальность (report_id, metric_def_id), валидация диапазона, перезапись значений.
- Возврат сохранённых метрик через API, корректная сериализация.

AC: (report_id, metric_def_id) уникально, [1..10]. Зависимости: S1.
