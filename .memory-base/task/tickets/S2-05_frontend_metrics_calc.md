Код: S2-05 — Frontend: Метрики и расчёт

Цель
- UI для ввода/правки метрик, кнопка «Рассчитать», отображение результатов.

Описание
- Экран отчёта: ввод/редактирование значений метрик (локаль ru‑RU, запятая), выбор профдеятельности, кнопка «Рассчитать», отображение результата и секций strengths/dev areas.

Шаги
- Формы с валидацией, форматирование чисел, вызовы API расчёта и отображение ответа.

Тестирование (обязательно)
- Валидация ввода, корректное преобразование запятая↔точка, обработка ошибок API.

AC: локаль ru‑RU (запятая), ошибки валидации. Зависимости: S2-04.
