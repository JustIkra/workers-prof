Код: S1-07 — Загрузка/скачивание отчётов (.docx)

Цель
- Принимаем `.docx`, сохраняем `file_ref`, отдаём исходник по API.

Описание
- Эндпоинт загрузки `.docx` привязывает файл к участнику и создаёт запись `report` с ссылкой на `file_ref`. Хранение локально в volume по стандартизированному пути; скачивание оригинала через защищённый стрим с `ETag`/`Content-Disposition`.

Шаги
- Проверка MIME/размера, путь `reports/{participant}/{report}/original.docx`.
- Скачивание с `ETag`.

Тестирование (обязательно)
- Загрузка корректного/некорректного MIME/размера, поведение при дубликате.
- Скачивание доступно авторизованным; ETag соответствует содержимому.

AC: 413 при превышении размера; 403 без доступа. Зависимости: S1-06.
