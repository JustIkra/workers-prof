Код: S3-02 — Stub извлечения

Цель
- `POST /reports/{id}/extract`, `GET /reports/{id}/metrics` с заглушками.

Описание
- Эндпоинт запуска извлечения создаёт задание (или выполняет stub‑логику) и переводит отчёт в статус. Эндпоинт чтения возвращает сохранённые метрики или предзаданные stub‑значения до интеграции AI.

Шаги
- Реализовать переход статусов, хранение/возврат текущих метрик, базовые ошибки.

Тестирование (обязательно)
- Повторный запуск не нарушает идемпотентность; сохранение ручной правки отражается в выдаче.

AC: консистентные значения, ручная правка сохраняется. Зависимости: S3-01.
