Код: S1-03 — Settings и профили dev/test/ci/prod

Цель
- Чёткие профили конфигурации; тесты — офлайн/детерминированные.

Описание
- Настроить Pydantic Settings со свитчем профилей окружения. В `test/ci` профилях включается детерминизм (замороженное время/seed), запрет внешней сети и `task_always_eager` для Celery. В `prod` — интеграции по реальным переменным.

Шаги
- Pydantic Settings с профилем.
- Тест‑профиль: Celery eager, внешняя сеть off.

Тестирование (обязательно)
- Проверить переключение профилей через переменную окружения и влияние флагов.
- Убедиться, что в `test` внешние вызовы замоканы/запрещены.

AC: Профили переключаются env‑переменной. Зависимости: S1-01.
