Код: S2-04 — Итоговый отчёт (JSON + HTML)

Цель
- Endpoint JSON по схеме + HTML шаблон (версионированный).

Описание
- Эндпоинт, возвращающий итоговый отчёт в строгом JSON формате (как в документации), и HTML‑рендер по версионированному шаблону. Значения/округления согласованы с сервисом расчёта.

Шаги
- JSON схема и сериализация; HTML шаблон; добавление версии шаблона в ответ.

Тестирование (обязательно)
