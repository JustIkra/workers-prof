Код: S1-10 — Раздача SPA и fallback

Цель
- Сборка фронта и раздача через FastAPI; fallback на `index.html`.

Описание
- Собирать фронтенд (Vite) и копировать `dist/` в контейнер приложения. FastAPI раздаёт статические файлы, а все неизвестные пути, кроме `/api`, отдаются как `index.html` для SPA‑маршрутизации.

Шаги
- `StaticFiles` + маршрутизация `/api` vs SPA.

Тестирование (обязательно)
- Проверить, что прямой переход на вложённые пути SPA открывает приложение (не 404).
- Проверить раздачу статики с корректными заголовками (cache/control по умолчанию).

AC: все не‑`/api` пути — SPA. Зависимости: S1-02.
