Код: SEC-01 — Обращение с секретами

Цель
- Не логировать секреты/PII; маскирование; хранение ключей вне VCS.

Описание
- Политика обращения с чувствительными данными: исключение секретов и ПДн из логов, маскирование значений в конфигурации/трассах, хранение ключей Gemini и приватных ключей WireGuard вне репозитория.

Шаги
- Маскирование секретов в логах; проверка репозитория на отсутствие приватных данных; регламент для операторов.

Тестирование (обязательно)
- Проверка логов на отсутствие секретов/ПДн; скан проекта на наличие ключей.

AC: аудит логов и конфигов; проверка отсутствия секретов в репо. Зависимости: S1.
