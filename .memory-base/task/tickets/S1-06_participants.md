Код: S1-06 — Participants CRUD + поиск

Цель
- Управление участниками и стабильная пагинация/сортировка.

Описание
- CRUD для участников с полями ФИО, дата рождения (опц.), внешний ID. Поиск по ФИО/ID, пагинация, сортировка по `full_name, id` для детерминизма. Аудит создающих/изменяющих действий по возможности.

Шаги
- CRUD, поиск по `full_name/external_id`.
- Сортировка `full_name, id`.

Тестирование (обязательно)
- CRUD операции, проверка пагинации/сортировки при совпадающих значениях.
- Фильтрация по подстроке/регистру для ФИО и точному совпадению ID.

AC: фильтры и пагинация детерминированы. Зависимости: S1-05.
