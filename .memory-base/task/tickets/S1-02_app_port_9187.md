Код: S1-02 — Порт 9187, прокси‑заголовки

Цель
- Uvicorn слушает `0.0.0.0:9187`, корректная работа за NPM (`--proxy-headers`).

Описание
- Приложение стартует на одном HTTP‑порте `9187` внутри контейнера. NPM терминирует TLS и проксирует HTTPS‑трафик, передавая `X‑Forwarded‑*`. Включаем поддержку прокси‑заголовков, чтобы адреса и схемы в ссылках и куки были корректными.

Шаги
- Настроить порт из `APP_PORT`.
- Включить `--proxy-headers`, `FORWARDED_ALLOW_IPS=*`.
- `/api/healthz` для health‑checks.

Тестирование (обязательно)
- Проверить доступность `/api/healthz` и корня `/` на `:9187` локально.
- Проверить корректную схему `https` за NPM (редиректы/куки не ломаются).

Acceptance Criteria
- `/` отдаёт SPA, `/api/healthz`=200.

Зависимости: S1-01.
