Код: AI-04 — Vision fallback (изображения)

Цель
- `extract_metrics_from_image()` со строгой фильтрацией токенов и диапазонов.

Описание
- Использование Gemini Vision как резервного канала для сложных изображений таблиц/бар‑чартов. Извлекаются только числовые значения в допустимых диапазонах с фильтрацией шума (символы «+/-/%», осевые подписи и пр.).

Шаги
- Промпт для извлечения чисел, пост‑валидация диапазонов и фильтров, возврат структуры значений с confidence.

Тестирование (обязательно)
- Проверка фильтрации «++/+/−/--», игнор оси 1..10, отбрасывание значений вне диапазона.

AC: «++/+/−/--» игнорируются, ось 1..10 фильтруется. Зависимости: AI-02.
