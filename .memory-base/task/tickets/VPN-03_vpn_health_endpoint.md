Код: VPN-03 — `/api/vpn/health`

Цель
- Возвращать состояние интерфейса, маршрутов, пробу до Gemini.

Описание
- Эндпоинт для оперативной диагностики VPN: статус интерфейса (up/down), список пиров/маршрутов и результат проверки доступности доменов Gemini через туннель.

Шаги
- Сбор данных об интерфейсе и маршрутах; попытка HTTP‑запроса к Gemini; формирование JSON‑ответа со статусом.

Тестирование (обязательно)
- Проверка ответов при активном и неактивном VPN; корректные коды (200/503).

AC: 200 при рабочем VPN; 503 при падении. Зависимости: VPN-02.
