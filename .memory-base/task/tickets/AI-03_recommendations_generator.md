Код: AI-03 — Генерация рекомендаций

Цель
- Промпт по схеме, self‑heal JSON при невалидной отдаче.

Описание
- Сборка промпта и системных инструкций для получения структурированных рекомендаций на русском языке по жёсткой JSON‑схеме. Повторный запрос/исправление при невалидном JSON (self‑heal) с ограничением длины списков.

Шаги
- Схема ответа, шаблон промпта, валидация/повторная попытка при ошибке парсинга.

Тестирование (обязательно)
- Проверка валидации схемы, обрезки длины списков, корректной локали RU.

AC: валидный JSON ≤5 элементов/секцию. Зависимости: AI-02, S2-02.
