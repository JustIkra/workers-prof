Git и коммиты

- Ветки
  - main: стабильная; develop (опц.)
  - feature/<short-name>, fix/<short-name>, chore/<short-name>

- Коммиты (Conventional Commits)
  - feat: новая функциональность
  - fix: исправление бага
  - chore/build/docs/refactor/test/perf
  - scope по модулю: feat(api): add participants search

- PR
  - Описание, скрины/логи, чек‑лист тестов
  - Запрет прямых пушей в main
