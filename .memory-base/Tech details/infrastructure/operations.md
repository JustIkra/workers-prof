Операционка: логирование, метрики, безопасность

- Nginx Proxy Manager
  - TLS терминация; редирект HTTP→HTTPS; форвард на `api-gateway:8000`
  - Rate limiting (опц.): защитить от брутфорса /auth/login

- Логирование
  - API: structlog/loguru с кореляционным request_id
  - Workers: отдельные логи по задачам (ocr/normalize/vision)
  - Хранение логов на volume; ротация

- Метрики/трейсинг (опц.)
  - Prometheus + Grafana; экспортеры для API/Workers
  - Тайминги OCR/LLM, доля fallback, ошибки

- Бэкапы
  - Postgres: ежедневные дампы; ретеншн ≥ 14 дней
  - Файлы (LOCAL/MinIO): периодическая проверка целостности; копии на отдельный диск

- Безопасность
  - Пароли: минимум 6 символов; 2FA не требуется
  - JWT с разумным TTL и refresh (опц.)
  - CORS/CSRF политика
  - Ограничение доступа к ссылкам на файлы — только авторизованные
