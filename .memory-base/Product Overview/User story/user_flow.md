Пользователь заходит на сайт «Цифровая модель универсальных компетенций (ЦМУК)».
Для начала работы он регистрирует личный кабинет или входит в уже существующий.
После входа в систему пользователю доступны несколько вкладок (например, «Настройки», «Участники тестирования»).
В разделе участников можно добавить нового участника по ФИО и открыть его индивидуальную страницу.
На странице участника предусмотрена возможность загрузить и просмотреть три отчета.
Все три отчета имеют одинаковую структуру и различаются только набором переменных и выводами.
Файлы отчетов хранятся в формате .docx.
Из каждого отчета требуется извлечь данные из таблицы, представленной в виде изображения.
Из таблиц необходимо получить числовые показатели характеристик.
Далее пользователь выбирает профессиональную деятельность.
Для каждой профессиональной области предусмотрена весовая таблица, в которую передаются полученные числовые данные.
На основании этих данных по формуле рассчитывается коэффициент — значение в процентном диапазоне.
Полученный коэффициент используется для формирования характеристики работника:

выделяются сильные стороны,

определяются зоны для развития,

предлагаются рекомендации по обучению — например, подбор тренингов или программ развития, направленных на повышение профессиональной пригодности.
