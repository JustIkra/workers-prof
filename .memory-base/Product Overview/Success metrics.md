Метрики успеха продукта

- Точность извлечения
  - ≥ 95% корректно распознанных числовых полей на контрольном наборе
  - < 2% кейсов с ручной правкой после дообучения/настройки

- Время обработки
  - От загрузки .docx до готового результата: p95 ≤ 60 сек при локальном OCR
  - Очередь фоновых задач не растёт при 10 параллельных загрузках

- Надёжность
  - Доля успешно завершённых пайплайнов ≥ 99%
  - 0 потерянных файлов/артефактов, ссылки стабильны

- Использование
  - Ежемесячный актив пользователей (MAU)
  - Кол-во рассчитанных результатов на пользователя
